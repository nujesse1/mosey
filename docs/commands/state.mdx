---
title: state
description: Get the semantic tree of the current page.
---

```bash
weblens state
```

Returns the accessibility tree of the current page as structured JSON. Every interactive element gets a `ref` ID that can be used with `weblens do`.

## Output

```json
{
  "url": "https://example.com",
  "title": "Example Domain",
  "tree": [
    {
      "ref": "r1",
      "role": "main",
      "children": [
        {"ref": "r2", "role": "heading", "name": "Example Domain", "level": 1},
        {"ref": "r3", "role": "link", "name": "More information...", "href": "https://www.iana.org/domains/example"}
      ]
    }
  ]
}
```

## Tree Node Properties

| Property | Type | Description |
|---|---|---|
| `ref` | string | Stable ID for use with `weblens do` |
| `role` | string | ARIA role (link, button, heading, textbox, etc.) |
| `name` | string | Accessible name (visible text, aria-label) |
| `value` | string | Current value (for inputs, selects) |
| `href` | string | Link target (for links) |
| `level` | number | Heading level (for headings) |
| `checked` | boolean | Check state (for checkboxes, radios) |
| `children` | array | Nested child nodes |

## Examples

```bash
# Get full state
$ weblens state

# Extract just link names
$ weblens state | jq '[.. | select(.role?=="link") | .name]'

# Count interactive elements
$ weblens state | jq '[.. | select(.role? | IN("link","button","textbox"))] | length'

# Find a specific element
$ weblens state | jq '.. | select(.name?=="Submit")' 
```

## Notes

- Each call takes a fresh snapshot (reflects current page state)
- The snapshot is stored internally for use by `weblens diff`
- Empty/decorative nodes are stripped to reduce token count
