## Install

```bash
# Clone and link
git clone https://github.com/mosey/mosey
cd mosey
bun install
bun link
```

Requires [Bun](https://bun.sh) and a Chromium browser (Playwright installs one automatically).

## Your First Session

### Navigate to a page

```bash
$ mosey navigate https://news.ycombinator.com
{"url":"https://news.ycombinator.com","title":"Hacker News","status":200}
```

The first command auto-starts the background browser engine. No setup needed.

### Read the page state

```bash
$ mosey state
```

Returns a JSON accessibility tree with `ref` IDs on every element:

```json
{
  "url": "https://news.ycombinator.com",
  "title": "Hacker News",
  "tree": [
    {
      "ref": "r1", "role": "banner",
      "children": [
        {"ref": "r2", "role": "link", "name": "Hacker News", "href": "/"}
      ]
    },
    {
      "ref": "r3", "role": "main",
      "children": [
        {"ref": "r4", "role": "link", "name": "Show HN: Mosey", "href": "..."},
        {"ref": "r5", "role": "link", "name": "Ask HN: ...", "href": "..."}
      ]
    }
  ]
}
```

### Interact with an element

Use the `ref` ID from the state output:

```bash
$ mosey do r4
{"success":true,"action":"click","ref":"r4"}
```

### See what changed

```bash
$ mosey diff
{
  "changes": [
    {"type": "navigation", "from": "/", "to": "/item?id=12345"},
    {"type": "added", "ref": "r20", "role": "heading", "name": "Show HN: Mosey"}
  ]
}
```

### Compose with standard tools

```bash
# Get all link names on the page
mosey state | jq '[.tree[].children[]? | select(.role=="link") | .name]'

# Navigate and extract title in one pipeline
mosey navigate https://example.com && mosey state | jq '.title'
```

### Save a session for later

```bash
$ mosey session save hackernews
{"saved":true,"name":"hackernews","domains":["news.ycombinator.com"]}

# In a future conversation:
$ mosey session load hackernews
{"loaded":true,"name":"hackernews"}
```

### Stop the engine

```bash
$ mosey stop
{"stopped":true}
```

The engine also auto-stops after 5 minutes of inactivity.
