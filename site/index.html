Commands

Every command outputs JSON to stdout. Pipe with jq. Compose with unix tools. That's the whole interface.

navigate <url>

Load a page. Auto-starts the engine if it isn't running.

weblens navigate https://stripe.com/dashboard

state

Semantic accessibility tree of the current page. Every element gets a stable ref ID.

weblens state | jq '.tree'

do <ref>

Click, type, or select by ref ID. Pass --value for text input.

weblens do r7 --value "hello@test.com"

diff

What changed since the last action. Structured delta + network requests.

weblens diff | jq '.network'

session save|load

Persist cookies and storage. Resume sessions across conversations.

weblens session save stripe_prod

describe

Emit a ~150 token prompt snippet. Drop it into any agent's system prompt.

weblens describe >> .claude/CLAUDE.md

Philosophy

⚙

The Bitter Lesson

Don't scaffold intelligence into the tool. Build the thinnest primitive. Let the model reason. Every "smart feature" is tech debt that the next model makes obsolete.

〉

Bash Is All You Need

The OS is already the agent runtime. MCP reinvents POSIX. We just add one more command to your shell — weblens — and the web becomes another pipe-friendly resource.

▣

Shared Window

The viewer isn't translating between agent and human realities. It's a shared window into the same web — where the agent reads structure and the human reads pixels, in real time.

⇅

Co-pilot, Not Autopilot

The human can click in the viewer at any time. The agent pauses, receives the action as context, and continues. No handoff ceremony. Just shared control.
